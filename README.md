## 简介

[Lime **(青柠记账)**](https://github.com/feiyu-rs/lime) 是一款开源的记账应用，采用目前主流的跨平台技术 Flutter 进行开发，后端采用完全支持 TypeScript 技术的服务器端开发框架 Nestjs，数据库使用 PostgreSQL，前后端接口采用 GraphQL技术...

## 项目结构

~~~
lime
├── android                                                 // 安卓端
├── ios                                                     // ios 端
├── windows                                                 // windows 桌面端
├── macos                                                   // mac 桌面端
├── linux                                                   // linux 桌面端
├── web                                                     // web 浏览器端
│
├── lib                                                     // Lime 主项目文件
│       └── 
~~~
